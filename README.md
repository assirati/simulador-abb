# Visualização de Árvores de Busca Binária

Visualizador interativo de Árvores de Busca Binária

> [!NOTE] > **Repositório de Tradução (PT-BR)**
>
> Este é um _fork_ do projeto original [bst-visualization](https://github.com/tjkendev/bst-visualization) de [tjkendev](https://github.com/tjkendev), focado na tradução da documentação para Português Brasileiro.
>
> Todo o crédito pelo código e pela ferramenta pertence ao autor original.

## Visão geral

Este projeto apresenta a visualização, em uma página web, das seguintes estruturas de dados:

- Árvore de Busca Binária
- Árvore AVL
- Árvore AVL Fraca
- Árvore Rubro-Negra
- Árvore Rubro-Negra Top-Down
- Árvore Rubro-Negra Esquerdistas
- Árvore Splay
- Árvore Splay Top-Down
- Árvore Scapegoat
- Treap
- Árvore de Busca Binária Aleatória

Você pode acessar a versão original em funcionamento neste link:

- [https://tjkendev.github.io/bst-visualization/](https://tjkendev.github.io/bst-visualization/)
